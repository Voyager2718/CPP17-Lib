# ytime
Location: [ytime](./ytime)

## Dependancy
- STL (iomanip, ctime, sstream)
- C++11 or above

## Namespace
- ytime

## Usage
- `std::string ytime::GetTime()`: GetTime returns formatted date and time in string. Example: **Mon Jul 10 12:29:15 2017 CST**.
